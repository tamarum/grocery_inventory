# Wrapup Skill

End-of-session housekeeping.

## Steps

1. Run full test suite: `cargo test --all-features`
2. Run lints: `cargo clippy --all-features -- -D warnings && cargo fmt --check`
3. Check git status: `git status`
4. Review uncommitted changes and suggest commits
5. Update CLAUDE.md if architecture changed
6. Note any TODOs or incomplete work for next session
